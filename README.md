# ETF_analysis

This project has a [wiki](https://github.com/TorontoDataScientistsWithoutBorder/ETF_analysis/wiki) for theory and stuff that is too big to fit in a readme.

I am using virtualenv for this project to ensure consistency across environments and (hopefully) make setting up/using/contributing a little easier. You can read a bit about python and virtualenv [here](http://docs.python-guide.org/en/latest/dev/virtualenvs/).

## Setup 
Using a unix-ish shell of your choice:

```
jupyter labextension install jupyterlab_bokeh
pip install virtualenv;
virtualenv env && source /env/bin/activate;
pip install -r requirements.txt
```
